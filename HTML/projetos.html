Case studies

2019 - Atual

Optical Logic Gates - Quality Analysis

This project aims to develop optical logic gates using the Interferometer Mach-Zehnder (MZI) in conjunction with a Semiconductor Optical Amplifier (SOA) and polarization control. The objective is to improve the quality of logic gates through simulations and experiments.

2019 - Atual

Magnetic Field Sensor - Ferromagnetic Fibers

This project seeks to create a magnetic field sensor using electromagnetic interference. modes in optical fiber coated with ferromagnetic materials, such as steel wool. The sensor is powered by a 1550 nm laser and can be applied to various areas.

2018 - Atual

Sagnac Interferometer Study - Optical Logic Gates

This project aims to develop an electric current sensor using the Sagnac interferometer in optical fiber coated with carbon steel. The sensor is powered by a 1550 nm laser and can be applied in various applications industrial.
